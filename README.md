## Upload
1. Back up your current repository.
2. Upload all files and folders from this package to the repository root.
3. Keep `CNAME` in the root for the custom domain.
4. In GitHub: Settings → Pages → Deploy from branch → `main` / root.
5. Test every page, phone link, email link and form after publishing.

## Important
The forms use `mailto:` and open the visitor's email app. For reliable web-form delivery, connect Formspree, Netlify Forms or another secure form provider later.

Replace any wording that does not exactly match your approved NDIS registration scope or current service capacity before publication.
