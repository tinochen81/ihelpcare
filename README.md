# iHelp Care Services Final V3

GitHub Pages-ready multilingual website.

## Upload
1. Download and unzip this package.
2. In your GitHub repository, remove the old website files after making a backup.
3. Upload **all files and folders inside this package** to the repository root.
4. Keep `CNAME`, `.nojekyll`, the `assets` folder and all HTML files.
5. Commit the changes and wait a few minutes for GitHub Pages to refresh.

## Included
- English, Japanese and Simplified Chinese switcher
- Gold Coast and Logan service areas
- Real hobby photography
- Core NDIS supports and shared-interest supports
- Facebook and Community Care SAKURA links
- Mobile-responsive design

## Important
Forms use `mailto:` and open the visitor's email application. Review all service wording against your current registration scope, qualifications, insurance, risk processes and service capacity before publication. Obtain written consent before using identifiable participant or family photos.
